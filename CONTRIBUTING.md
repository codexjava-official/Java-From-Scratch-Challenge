# Contributing to Java From Scratch Challenge

Welcome! We are thrilled that you want to contribute to the **CodeXJava Official** community workspace. By participating in this 100-day developer challenge, you agree to abide by our Code of Conduct.

---

## ⚡ How Can I Participate?

To keep the workspace clean and maintain an effective review process, please follow this step-by-step contribution lifecycle:

### 📦 Submitting Your Daily Challenge Solutions

#### 1. Fork the Repository
Click the **Fork** button at the top-right corner of this main page to create an isolated copy of this project under your personal GitHub account.

#### 2. Clone Your Fork
Download your newly forked repository locally to your machine by opening your terminal or Git Bash and running:
```bash
git clone [https://github.com/codexjava-official/Java-From-Scratch-Challenge.git](https://github.com/codexjava-official/Java-From-Scratch-Challenge.git)
