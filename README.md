# 💻 Coding Standards

Maintain professional coding practices.

## Naming Conventions

### Variables

```java
int studentCount;
double averageScore;
```

### Methods

```java
calculateTotalMarks();
printStudentDetails();
```

### Classes

```java
Student
EmployeeManager
BankAccount
```

---

## Avoid Magic Numbers

❌ Bad

```java
salary = salary + 5000;
```

✅ Good

```java
final int BONUS_AMOUNT = 5000;
salary += BONUS_AMOUNT;
```

---

## Import Only Required Classes

❌ Bad

```java
import java.util.*;
```

✅ Good

```java
import java.util.Scanner;
import java.util.ArrayList;
```

# 📝 Solution Submission Format

Create your workspace inside the day's folder.

Example:

```text
Day-01_Introduction
└── your-github-username
    └── Solution.java
```

Example code structure:

```java
/**
 * Day 01 Challenge
 * Author: @your-github-username
 * Time Complexity: O(1)
 * Space Complexity: O(1)
 */

public class Solution {

    public static void main(String[] args) {

        long transactionId = 9876543210L;
        char assetClass = 'A';
        double indexRate = 845.67;

        int truncatedValue = (int) indexRate;

        System.out.println("Transaction ID : " + transactionId);
        System.out.println("Asset Class    : " + assetClass);
        System.out.println("Index Rate     : " + indexRate);
        System.out.println("Converted Rate : " + truncatedValue);
    }
}
```
